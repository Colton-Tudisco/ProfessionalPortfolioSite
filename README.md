# Professional Portfolio Site
A personal portfolio website built to showcase my academic projects, technical skills, and professional experience. Built with a mobile-first approach using semantic HTML5 and CSS3.
 
## Pages
- **Home** (`index.html`) — Landing page with intro and welcome message
- **About** (`about.html`) — Bio, goals, and background
- **Projects** (`projects.html`) — Showcase of development projects
- **Resume** (`resume.html`) — Education, experience, technical skills, and references
- **Contact** (`contact.html`) — Contact information and links
## Tech Stack
- HTML5 (semantic elements)
- CSS3 (mobile-first, Flexbox nav, Grid layout, media queries)
- Git / GitHub for version control
