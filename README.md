# Professional Portfolio Site
A personal portfolio website built to showcase my academic projects, technical skills, and professional experience.

## Pages
- **Home** (`index.html`) — Landing page with intro tagline
- **About** (`about.html`) — Bio and background
- **Projects** (`projects.html`) — Showcase of development projects
- **Resume** (`resume.html`) — Education, experience, technical skills, and references

## Tech Stack
- HTML5
- CSS3 (mobile-first, custom color palette)
